# PROJECT_SUMMARY вҖ” PART 1

## Project description

**Ollie** вҖ” an app that helps children learn to read and explore STEAM topics (Science, Technology, Engineering, Arts, Mathematics). Children talk to **Ollie**, an AI tutor that speaks like a renowned teacher trained to handle severe ADHD. Target audience is **Kindergarten through 6th grade**. Strict child-safe content filtering: no words inappropriate for children. Several sections for children (Learn, Exercise, Spell, Progress) plus a PIN-gated parent settings dashboard.

Surfaces in scope: web (mobile + desktop browsers via Vercel), Android (Capacitor 7), iOS (Capacitor вҖ” config stub only, native project not yet scaffolded), Vercel-hosted serverless backend.

Solo developer. Production-quality bar per `CLAUDE.md` (UAT-gated done, behavior coverage, cross-platform safe areas, 99% uptime SLO, centralized observability, reversible deploys).

### Server modules

**Vercel serverless (`api/`, CommonJS):**

- `api/ai/generate.js` вҖ” POST. Gemini text-gen proxy. Primary `gemini-2.5-flash` Г—3 with exp backoff, fallback `gemini-2.0-flash`. Injects `BLOCK_LOW_AND_ABOVE` safety on all 4 categories.
- `api/ai/speak.js` вҖ” POST. TTS via `gemini-2.5-flash-preview-tts`. Pre-rejects text containing `[`, `]`, `(`, `)`, `/` (model returns 400 "should only be used for TTS"). Returns `{ audio, mimeType }`.
- `api/ai/live-token.js` вҖ” POST. Builds Gemini Live WebSocket URL with the API key in the query string and returns it. **The client receives the key in the URL.**
- `api/health.js` вҖ” GET. Pings `gemini-2.5-flash` with a 5-token "ping". 8s abort timeout, 3 retries with backoff on 429/503.
- `api/errors.js` вҖ” POST. Client error sink вҶ’ `console.error`, returns 204.

**Local Express proxy (`server/index.js`):** binds `0.0.0.0:3456`. Reads `.env` by hand (no dotenv). Implements **only** `GET /health` and `POST /ai/generate` вҖ” does not implement `/ai/speak` or `/ai/live-token`, so it is **not a drop-in replacement** for the Vercel deployment.

### Frontend (`www/`, Capacitor `webDir` + Vercel `outputDirectory`)

- `www/js/countdown.js` вҖ” reusable 60s SVG-ring talk-window timer (Learn/Exercise/Spell mics); honors `prefers-reduced-motion`.
- `www/js/spell-tools.js` вҖ” pure spelling utilities: `cleanSpellWord`, `cleanSpellMeaning`, `buildSpellResult`, `parseSpellLookup`, `extractSpellTarget` (regex-detects "spell X" / "how do you spell X", ignores pronouns), `buildSpellTeachMessage`. - `www/js/speech.js` вҖ” Gemini Live WebSocket client. 24kHz `AudioContext` mic capture, audio playback queue, transcript accumulation, listen-window timer, per-tab mic toggles. `LIVE_MODELS = ['models/gemini-3.1-flash-live-preview']`. Also handles REST TTS fallback path (`/ai/speak`).
- `www/js/ai.js` вҖ” REST AI engine. SessionStorage prompt cache (10-min TTL, SHA-256 keyed). Per-day token usage tracking. On-device Gemma probe. `aiGenerate()`, `sysPmt()` (Ollie persona вҖ” STEM teacher, voice-only, ENGLISH ONLY, K-6 rules, mode tones: normal/excited/calm/night), `sysPmtJSON()`, `callAI()` learn loop, `recordLearnActivity()`, `rememberConversationTurn()`. Spell-target detection short-circuits to deterministic spell flow.
- `www/js/exercises.js` вҖ” `EXSYS` and `CHKSYS` system prompts, math sub-skill picker (Counting/Adding/Subtracting/Multiplying/Dividing) with KвҖ“6 ladder hints, MCQ + fill-blank + voice-answer rendering, `submitFB`/`finishEx`/`nextEx`.
- `www/js/progress.js` вҖ” Progress tab: per-subject activity bars, accuracy, badges, streak, recent-activity feed (interleaves exercise вң…/вқҢ, learn рҹ’¬, spell рҹ”Ө).
- `www/js/ui.js` вҖ” tab switching, mode/difficulty/listen-wait setters, grade picker, parent PIN flows (`chkPin`, `openDash`, `openChgPin`, `saveChgPin`), spell tab handlers (`spellWord`, `togSpellMic`, `_spellHistory` вҶ’ `kai5_spell`), typed-message Learn flow, COPPA accept gate.
